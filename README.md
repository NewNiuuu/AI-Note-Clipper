- # ✂️ AI-Note-Clipper (AI 笔记提炼器)

  一个轻量级的 Edge/Chrome 浏览器扩展。让你在浏览网页（尤其是与 AI 助手的冗长对话）时，能够一键提取文本的**核心信息**，并自动复制到剪贴板。

  <img src=".\img\using example.png" alt="using example" style="zoom: 67%;" />

  ## 💡 为什么要做这个项目？

  这个项目的诞生源于一个真实日常痛点：

  在使用大语言模型时，常常能得到非常详尽、长篇大论的优质回答。这很好，但当我们已经理解了这些内容，想要将“我的问题”和“AI的回答”记录到个人笔记软件中作为**”快速记忆索引“**时，冗长的文本反而成了负担。

  我想要建立的是一个**“记忆索引”**，而不是一本厚厚的百科全书。我只需要记录提问的核心，以及回答的最关键结论。

  因此，我用claude code开发了这个插件：**零繁琐操作，划选文本 -> 右键提炼 -> 自动复制极简 Q&A。** 整个过程如丝般顺滑，不打断原有的阅读与思考。

  ## 🚀 核心特性

  - **无缝体验：** 无需唤出复杂的界面，基于右键菜单和系统通知完成闭环。
  - **配置自由：** 内置可视化设置面板（Popup），轻松配置你自己的 API Key（默认支持 DeepSeek 等兼容 OpenAI 格式的接口）。
  - **零复制粘贴：** 提炼完成后自动写入系统剪贴板，并伴随优雅的网页 Toast 提示。
  - **纯粹轻量：** 100% 原生 HTML/CSS/JavaScript 构建，无任何臃肿框架，极其丝滑。
  
  ## 🛠️ 高度客制化与拓展潜力

  这个插件虽然叫做“AI 笔记提炼器”，但由于我们将“思考的大脑”交给了 API，并在设置页开放了 **自定义 Prompt（提示词）** 的输入，**这使得它本质上成为了一个「网页任意文本的万能处理工作流」。**

  <img src=".\img\extension config.png" alt="extension config" style="zoom: 67%;" />

  你可以通过修改 Popup 界面中的 Prompt，将其零成本改造为任何你需要的工具：

  * **📝 行动清单提取器：** > *Prompt 示例：请从选中文本中提取出所有的待办事项，以 "- [ ]" 的格式输出。*
  * **🌍 网页极速翻译机：** > *Prompt 示例：你是一个翻译专家，请将选中文本翻译为优雅的中文，仅输出翻译结果。*
  * **💻 晦涩代码解释器：** > *Prompt 示例：请用一句话大白话解释这段代码的作用，并在下面列出它涉及的核心函数。*
  * **👔 职场黑话翻译器：**
    
      > *Prompt 示例：请帮我把这段客套的商业邮件转化为最直接的大白话，告诉我对方到底想干嘛。*
  
  你的想象力，就是这个工具的边界！

  ## 📦 安装说明 (本地加载)

  由于本项目仍在迭代中，目前可以通过“开发者模式”本地安装：

  1. `git clone` 或下载本仓库代码到本地文件夹。
  2. 打开 Edge 或 Chrome 浏览器，进入扩展管理页面 (`edge://extensions/` 或 `chrome://extensions/`)。
  3. 开启右上角的 **“开发人员模式”**。
  4. 点击 **“加载解压的扩展”**，选择本项目的文件夹即可。
  5. 点击浏览器工具栏的插件图标，填入你的 API Key，开始体验！
